Fundamentos informática

Conceptos clave

Internet

Concepto

Es un conjunto descentralizado de redes de comunicaciones interconectadas, que utilizan la familia de protocolos TCP/IP, lo cual garantiza que las redes físicas heterogéneas que la componen constituyen una red lógica única de alcance mundial.

Dominios

Es un nombre único que identifica a una subárea de Internet, en otras palabras se encarga de traducir las direcciones IP de cada activo en la red, a términos memorizables y fáciles de encontrar.

Hipertexto

Es una estructura no secuencial que permite crear, agregar, enlazar y compartir información de diversas fuentes por medio de enlaces asociativos y redes sociales

Protocolos

Los protocolos para la transmisión de datos en internet más importantes son TCP (Protocolo de Control de Transmisión) e IP (Protocolo de Internet). De manera conjunta (TCP/IP) podemos enlazar los dispositivos que acceden a la red, algunos otros protocolos de comunicación asociados a internet son POP, SMTP y HTTP.

Direcciones

Formada de dos partes de una dirección Internet son la parte de la dirección de red y la parte de la dirección del sistema principal. Esto permite que un sistema remoto especifique tanto la red remota como el sistema principal en la red remota al enviar información. Por convenio, se utiliza el número de sistema principal 0 para hacer referencia a la propia red.

Servicios Internet

Correo electronico

Es un servicio de red que permite a los usuarios enviar y recibir mensajes mediante redes de comunicación electrónica.

Transferencia archivos FTP

FTP es el conjunto de reglas que los dispositivos de una red TCP/IP (Internet) utilizan para transferir archivos. Cuando usas Internet, en realidad utilizas una variedad de diferentes protocolos. Para navegar, usas HTTP. Para enviar y recibir mensajes instantáneos, utilizas XMPP. El FTP es simplemente el protocolo usado para trasladar archivos.

World Wide web

La World Wide Web o red informática mundial​ es un sistema que funciona a través de internet, por el cual se pueden transmitir diversos tipos de datos a través del Protocolo de Transferencia de Hipertextos o HTTP, que son los enlaces de la página web.

Formas de conexion

Existen muchas formas de conectarse a web, unas que ya estan obsoletas por su poca eficiencia en el año actual y otras estan surgiendo a nivel global por ser las mas fiables a nivel de velocidad se trata, por ejemplo antes se utilizaban cables de telefono para realizar las conexiones a internet pero era muy lento ya que se manejaban velocidades maximas de 56kbps, en cambio hoy en dia se manejan velocidades de hasta 100Mbps.

Proveedores

Proveedor de internet se conoce como a la empresa la cual se encargara de brindarle conexion a internet y demas servicios.

Conexion por cable

Software de acceso

Es un programa que se codifica en un lenguaje interpretable por los navegadores web en la que se confía la ejecución al navegador. Un claro ejemplo son los navegadores que se conectan los navegadores web o a una intranet.
